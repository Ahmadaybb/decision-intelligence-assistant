
# Decision Intelligence Assistant

A RAG-powered customer support assistant with ML and LLM priority prediction.

## Features
- Retrieval-Augmented Generation for context-aware answers
- ML baseline (Random Forest) for priority prediction
- LLM zero-shot priority prediction
- Side-by-side comparison of all approaches

## Setup
```bash
docker compose up --build
```

Access at: http://localhost:3000
